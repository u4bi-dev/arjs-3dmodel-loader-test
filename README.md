> ### [clara.io](https://clara.io) 
> 0. SignUp / Login
> 1. New Scene
> 2. Create Empty Scene
> 3. [Create]
> 4. File -> Import Files -> upload file -> [Import]
> 5. File -> Export All -> ThreeJS(.json)
